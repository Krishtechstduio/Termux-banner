# Termux-banner
A best tool to make banner in termux

1.)`pkg install git`

2.)`git clone https://github.com/Krishtechstduio/Termux-banner`

3.)`cd Termux-banner`

4.)`chmod +x *`

5.)`./requirement.sh`

6.)`./banner.sh`
